get_eco

specifies ecology for AmP entries

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

[climate, ecozone, habitat, migrate, food] = get_eco(taxon)

Description

Fills cell strings with ecological specifications of AmP species

Input:

  • taxon: character string with name of an entry

Output:

  • climate, ecozone, habitat, migrate, food: cell arrays with codes

Remarks

codes are described in AmPeco.html

Example of use

[climate, ecozone, habitat, migrate, food] = get_eco('Daphnia_magna');
